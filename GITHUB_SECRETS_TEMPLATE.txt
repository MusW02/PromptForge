WHAT HAPPENS AFTER SECRETS ARE ADDED
================================================================================

2. Go to: github.com/USERNAME/prompt-generator/actions
3. Watch the workflow run:
   ✅ Tests run
   ✅ Deploy to Heroku
4. Once complete, your app lives at:
   https://promptforge-yourname.herokuapp.com

================================================================================
IF SOMETHING GOES WRONG
================================================================================

Check Heroku logs:
```bash
heroku logs --tail -n 100
```

Looking for errors? Common ones:
- "ModuleNotFoundError" → Missing package in requirements.txt
- "Connection error" → Can't reach Ollama API
- "KeyError: OLLAMA_API_KEY" → Secret wasn't set correctly
- "500 Internal Server Error" → Check the logs above

Fix issues locally, test with pytest, then push again:
```bash
pytest tests/ -v
git add .
git commit -m "Fix deployment issue"
git push origin main
```
